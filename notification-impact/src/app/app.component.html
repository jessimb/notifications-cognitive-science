Welcome to Emotifications, the notification stress model

With this tool, you can simulate the effect of notifications on stress levels. This tool was built for Georgia Tech's CS6795.

Instructions

Enter in a schedule of notifications on the right panel. Add as many entries as you'd like. Click "Simulate" to run the model with the provided schedule. You can display multiple runs from the model on the graph, or clear the output between runs. You can zoom into a portion of the chart using the timeline below the chart. You can also export the chart as an image.

  • Type: Notification type
  • Start time: When the notifications should begin being sent
  • Count: How many notification to send in total
  • Interval: How far apart to send notifications

About the model

The model assumes the schedule to the right. For example, from midnight to 8:30am, the modeled person has caretaking responsibilities.

Notifications can only be sent one at a time. If you want to send multiple notifications at the same time, you can add a new entry that begins at the same time.

The model was generated with the following equation:

stress = stress level due to notification type × stress level when receiving a notification in a certain context × stress level due to notification frequency

We accounted for stress decay over time by reducing stress by each notification's time-to-live.

stress = stress × (time-to-live – age / time-to-live)

The values for stress levels and time-to-live were derived from a survey of 100 people.
